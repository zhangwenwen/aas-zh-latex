# aas-zh-latex
chinese latex template for aas 

## Environments
 OS: macos Big Sur 11.3.1

 Tex: texlive 2019

Editor: texstudio 3.1.1
 ## How to use

1. setup the environment
2. double-click for fonts installation from the directory fonts/
3. compile the tex file with xelatex
